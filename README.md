## Featured Projects (built with Claude Code)

### 🛒 EcomMind AI — E-commerce Support Agent
Intelligent 24/7 multi-platform e-commerce support agent for **Shopify, Amazon, TikTok Shop, and Facebook Marketplace**. Upload product catalogs, policies, and FAQs into knowledge bases (PDF/CSV/Excel/TXT/MD/JSON) with hybrid vector + full-text search (Reciprocal Rank Fusion); the agent answers with citations and live tools for order lookup, inventory checks, returns, and shipment tracking. Multi-tenant SaaS with an LLM router (OpenAI / Anthropic / Groq / Gemini / Ollama) with automatic fallback, admin chat, customer-facing chat, settings, and analytics dashboards.

**Stack:** Next.js 15 (App Router) · TypeScript · LangGraph · Vercel AI SDK · Supabase (Postgres + pgvector) · Clerk · Tailwind + shadcn/ui

### 📞 AI Receptionist SaaS
Multi-tenant voice AI receptionist. Business owners forward their existing phone number to a shared Vapi number; the system identifies the client from the forwarding data and routes the call to that client's own AI agent — their script, voice, and Google Calendar booking. Call scripts are auto-generated by scraping the client's website with Firecrawl and Anthropic Claude.

**Stack:** Next.js (App Router) · TypeScript · Supabase (Postgres + Auth + RLS) · Vapi · n8n · Google Calendar API · Claude API

### 🎙️ EchoScribe — Offline Audio Recorder & Transcriber
Windows desktop app that records audio and transcribes it **100% offline** with faster-whisper — no cloud, no per-minute API costs. Includes an in-app Whisper model manager (download/switch models), a searchable recording library, transcript exports, and an Inno Setup installer.

**Stack:** Python · faster-whisper · Tkinter UI · Inno Setup

### 💻 Call Recorder (Desktop)
Cross-platform desktop call recorder built with Electron and React, using FFmpeg for audio capture and processing. Packaged with electron-builder for Windows (NSIS), macOS, and Linux (AppImage).

**Stack:** Electron · React · TypeScript · Vite · FFmpeg · Tailwind

### ⚡ ESID Scraper — EnergyBot → Google Sheets Automation
n8n workflow that watches a Google Sheet for new addresses, queries EnergyBot's premise-search API, validates every candidate against the input address (house number, street, zip/city) to reject fuzzy false-positives, and writes up to 8 verified ESID meter IDs back to the exact row. Includes per-row throttling, 3× retries with an error branch, and self-healing sweeps that pick up rows missed by failed runs.

**Stack:** n8n · Google Sheets API · REST · JavaScript
